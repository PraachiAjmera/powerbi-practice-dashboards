# Power BI Practice Dashboards

## Overview
This repository contains my Power BI practice dashboards created to improve my skills in data analysis, data cleaning, DAX, and report building.

The dashboards are created for learning purposes and cover different stages of Power BI, from basics to enhanced reporting.

## Dashboard Details
1. Basics
   - Data loading
   - Basic visuals
   - Simple reports

2. Data Cleaning
   - Data transformation using Power Query
   - Handling missing and incorrect data

3. DAX Practice
   - Calculated columns
   - Measures
   - Basic DAX functions

4. Performance Analytics
   - KPI analysis
   - Performance and trend analysis

5. Report Practice
   - Report layout
   - Visual selection
   - User-friendly design

6. Enhanced Report
   - Improved visuals
   - Better interactivity
   - Refined insights

## Tools Used
- Power BI Desktop
- Power Query
- DAX

## How to View
1. Download the .pbix file
2. Open it using Power BI Desktop

## Note
These dashboards are created using Microsoft Learn for learning and practice purposes to strengthen my Power BI and data analytics skills.
